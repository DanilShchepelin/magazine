## Запуск проекта

```bash
  # Создать и настроить файл .env
  cp .env.example .env
```

### Запуск проекта в Docker

#### Поднять проект и следить за приложением в консоли

```bash
  make start_live
```

#### Поднять проект в фоне

```bash
  make start
```

#### Выключить проект

```bash
  make down
```
